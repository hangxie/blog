It seems 3.0.3 is a good version, at least to me.<\/p>\n

I think I was one of the first batch that tried out TB3, however, it seems my mailbox does not fit into TB3 – I have ~600 mail folders, with ~110K mails and the biggest folder has about 8000 mails. Obvious old version of TB3 cannot handle this – it popped up error message window from time to time and every time I started TB3 I have to wait for pretty long time to get everything sync’ed.<\/p>\n

Thanks 3.0.3, it works now.<\/p>\n

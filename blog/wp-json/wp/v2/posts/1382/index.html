Time to play with Virutalization again, this time I’m using kvm on CentOS 6, which has pretty many documentation but do need to pay more attention on details – I was locked out of the box last night after changing the network setting (with mistakes for sure \ud83d\ude41 ), this should be avoided for anyone who’s going to do this. Again, stay beside your machine whenever you are to do the initial setup.<\/p>\n

After network setup is done, everything else is pretty smoothy, just need to read lots of man pages to know syntax of those qemu and virt-* commands. I repeated several times by virt-install, then virsh destroy and virsh undefine, etc., then virt-install again. There should be some better way to change configuration for existing VMs but for me, it’s much more easier to run from scratch.<\/p>\n

Now I have 4 VMs running on a 8-cores\/32G box, it seems everything’s OK so far, but I need to do some more tests – especially performance test – before carrying on.<\/p>\n

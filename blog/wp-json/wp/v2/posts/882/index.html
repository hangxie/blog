Since I’m getting myself messed up with all sort of distros, here are something I have to write down to keep track.<\/p>\n

To make Linux\/FreeBSD distro up-to-date:<\/p>\n

    \n
  • Debian & Ubuntu:
    \nalias update=’sudo apt-get -y update && sudo apt-get -y upgrade && sudo apt-get -y dist-upgrade’<\/li>\n
  • CentOS & Fedora:
    \nalias update=’sudo yum -y update’<\/li>\n
  • openSUSE:
    \nalias update=’sudo zypper refresh && sudo zypper –no-gpg-checks -n update’<\/li>\n
  • Gentoo:
    \nalias update=’sudo emerge –sync && sudo emerge –update –deep –newuse world’<\/li>\n
  • FreeBSD (need to have portmanager installed):
    \nalias update=’sudo portmanager -u’<\/li>\n<\/ul>\n","protected":false},"excerpt":{"rendered":"

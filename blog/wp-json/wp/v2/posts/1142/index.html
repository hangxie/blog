Got an MacBook Pro 13″, but it’s an old model as the newest models are still pending on “regulator’s approval” in China, per Apple’s web site. I believe it’s the problem from the new Thunderbolt.<\/p>\n

Mine has 8G memory so I can run 3 VMs (under VirtualBox, each has 1G memory) plus all applications from Microsoft Office (Word, Excel, PowerPoint, and Outlook), Safari, and several small utilities at the same time without any performance problem. For most time I only need to run 1 VM so it should be OK.<\/p>\n

VirtualBox rolls, while Parallele Desktop and VMWare Fusion both suck. Alright I know I’m not familiar with either so VirtualBox gained my respect, but thinking of VirtualBox is free, this simply kills those two.<\/p>\n

Installing Xcode was a nightmare – App Store in China didn’t work, at least for huge packages like Xcode (I installed some small stuffs without problem), finally I downloaded Xcode from Apple’s developer web site. There were not many to be installed as I can see, other than QQ and MPLayer. I’m trying to get Pidgin running now, but seems I have to run it under X11, and if this is the case I will move to Adium though I do want to have customized smileys. Maybe Adium got it already, need to check out.<\/p>\n

Overall speaking, MBP is a nice laptop, it’s quiet, cool (against HOT MBP ppc I had 8 years ago). The only problem is that I have to remove my watch while working on it as chain of the watch will scratch the shell.<\/p>\n

I ordered a keyboard screen and a bag for it, will get this week.<\/p>\n","protected":false},"excerpt":{"rendered":"

I got template not found etc error while playing with Django, and obviously my turtorial material is not up to date.<\/p>\n

Symptom is described here:<\/p>\n

http:\/\/sam.bluwiki.com\/blog\/2008\/07\/django-error-while-importing-urlconf.php<\/p>\n

“If you’re getting it, it’s likely you updated to the latest Django SVN release, and haven’t upgraded your admin site’s configuration.”<\/p><\/blockquote>\n

and the solution link is on the same page, which, in short, add following lines to urls.py:<\/p>\n

from django.contrib import admin
\nadmin.autodiscover()<\/p><\/blockquote>\n

and uncomment this line:<\/p>\n

(r’^admin\/(.*)’, admin.site.root),<\/span><\/p><\/blockquote>\n

that’s it …

Several things happened in the past a couple years so I moved this blog site to static, some major issues lead to this change:<\/p>\n

    \n
  • Google is no longer my preferred service provider, so I’m moving away from AdSense and Analytics<\/li>\n
  • I’m no longer actively writing so I don’t expect people interact with me through this site<\/li>\n
  • Google is going to stop G Suite legacy so I assume they will stop GCE free tier soon, which I’m running this site at<\/li>\n<\/ul>\n

    For now I have an VM running Ubuntu with WordPress installed, I launch the VM whenever I want to write something, then using Simply Static<\/a> to generate pure static pages then copy to an Nginx server. The Nginx is still running on GCE but since everything’s static, I can easily port it to anywhere cheap whenever needed.<\/p>\n
